# 採用広告

**採用広告**は、広告運用・開発に続いて最近始めた事業です。

## サービス内容

- **配信**: [Air Work](https://airwork.jp/) で有料広告を配信
- **PDCA**: クリック率・応募率などを見ながら**文案のPDCA**を回す
- **応募者対応**: 応募者とのやり取りは**内（メンテモ）**が実施。スケジュールを組むところまで対応
