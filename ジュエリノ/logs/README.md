# ジュエリノ logs

ジュエリノ（JEWELINO）に関する**サマリログ**を記録するフォルダです。

## 運用方針

- 雛形や全体像のドキュメントは `../全体像/` や各サブフォルダに保存
- この `logs/` には、**要点だけを抜き出したJSONL形式のログ**を保存
- 1行 = 1イベント（日時・種別・概要・関連ドキュメントへの相対パスなど）

## いま入っているもの

- `2026-02-24_ジュエリノ要件整理.jsonl`
  - JW事業の全体像ドキュメントを作成した際の要約ログ
  - `全体像/2026-02-24_ジュエリノ事業概要と運用要件.md` を指し示す
