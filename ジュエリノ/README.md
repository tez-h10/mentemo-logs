# ジュエリノ（JEWELINO）

自社事業（郵送でジュエリー修理を完結させるサービス）の資料を、役割ごとに整理するフォルダです。

## 構成

| フォルダ | 内容 |
|----------|------|
| **[全体像](./全体像/)** | 事業概要・Ops方針・KPI・運用要件など、開発・マーケ・Ops 共通の前提ドキュメント |
| **[マーケ](./マーケ/)** | KPI運用・広告評価・LP・施策メモ（マーケ担当が主に使用） |
| **[開発](./開発/)** | Admin / LINE 等の開発要件・仕様（開発担当向け。マーケは参照のみ） |
| **[logs](./logs/)** | 要件整理・変更履歴などのサマリログ（JSONL）。必要な情報のみ記録 |

**参照の目安**: 全体像は共通。マーケは主に **マーケ** と **全体像** を、開発担当は **開発** と **全体像** を参照してください。
