# 東日本エネルギー

**広告運用案件**です。出光興産グループ・株式会社東日本エネルギー様向け。**オートフラット**の集客（リスティング広告・Meta広告・アンケート活用）の提案・運用を進めています。

## 構成

- **data/** … 案件概要・ご提案資料の要点など必要なデータのみ（JSON）。
- **logs/** … 提案・打ち合わせ・施策メモなどのサマリ（JSONL）。

## 案件スコープ（要約）

- **商材**: オートフラット（カーリース等）
- **集客**: リスティング広告（顕在層）、Meta広告（準顕在層）、アンケート（店頭レシート・LINE・SNS）の結果をLP・バナー訴求に活用
- **スケジュール**: キックオフMTG→4月配信スタート想定、5月〜定例会で数値共有・改善

詳細は `data/オートフラットご提案_2026-02-26.json` を参照。SSIDは [Google広告運用/SSID一覧.md](../Google広告運用/SSID一覧.md) で確認（東日本エネルギーは SS00010 等）。
