# エザキ

**広告運用案件**です。クライアントはエザキ（株）。カーリースとKeeper（コーティング）の2本柱で集客を運用しています。

## 構成

- **data/** … 案件概要・施策メモなど必要なデータのみ（JSON）。ログは要点だけ。
- **logs/** … 概要更新・施策メモなどのサマリ（JSONL）。

## 案件スコープ（要約）

| 柱 | 流れ |
|----|------|
| カーリース | LP→問い合わせ/予約（LP稼働中） |
| Keeper（コーティング） | キャンペーン/訴求→問い合わせ/予約 |

媒体：Google検索（リスティング）＝顕在層・競合防衛、Meta＝準顕在層・創出。詳細は `data/案件概要.json` を参照。
