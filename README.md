# mentemo logs

会社では**広告運用**と**開発**が主軸です。このリポジトリは、そのログ・設定・要件を**案件・事業ごと**に整理するためのものです。

## 構成

| フォルダ | 内容 |
|----------|------|
| **[ピッカーズ](./ピッカーズ/)** | 広告運用案件。施策ログ・広告設定・議事録（マーケ・運用の拠点） |
| **[エザキ](./エザキ/)** | 広告運用案件。カーリース・Keeper（コーティング）の2本柱。概要・logs |
| **[ジュエリノ](./ジュエリノ/)** | 自社事業。全体像（共通）・マーケ・開発・logs |

マーケは施策・KPI・全体像を主に参照・更新。開発向けの要件はジュエリノ/開発にまとめます。

## 運用

- 案件ごとにフォルダを分け、中で施策ログ（data / logs）や広告設定・開発要件を管理
- **雛形は保存** … スクリプト・テンプレ・設定の型はリポジトリに残す
- **ログは必要な情報のみ** … 全ログは残さず、判断・参照に必要な要点だけ残す（容量を抑える）。メール文全文・毎回の生データは保存しない
- 日付は `YYYY-MM-DD` 形式
