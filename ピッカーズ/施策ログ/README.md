# ピッカーズ 施策ログ

ピッカーズは**広告運用案件**です。ここはその施策・データ・ログの置き場です。やり取りや運用の**必要な情報だけ**を残します。

## 運用方針

- **雛形は保存する** … スクリプト・テンプレ・設定の型はリポジトリに残す。
- **ログは必要な情報のみ** … 全ログを貯めず、判断・参照に必要な要点だけ残す（容量を抑える）。
- メール文全文・毎回の生データの蓄積はしない。

## フォルダ構成

- **data/** … 数値・リンク・マスタなど「必要なデータ」のみ（JSON 等）。
- **logs/** … 依頼・送付・会議などの**サマリ**（日付・種別・要点のみ。JSONL）。

日付は `YYYY-MM-DD` 形式で記録します。
