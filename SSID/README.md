# SSID（会社）別フォルダ

**Google広告の仕組み（SSID/LTID・GL→スプシ）** は会社（メンテモ）の仕組みです。ここでは **SSID ＝ 会社** ごとにフォルダを切り、その会社に紐づく LTID（キャンペーン＝エリア＋商材）を一覧できるようにしています。

## マスタの所在

- **ssid_ltid_mapping.json**: [Google広告運用/config/ssid_ltid_mapping.json](../Google広告運用/config/ssid_ltid_mapping.json)  
  ピッカーズと SSID のある案件は**並列**。会社共通の雛形・config はルート直下の **Google広告運用** にあります。
